Anal
====

Please execute it.
``` bash
	$ git clone https://github.com/5st7/Anal
	$ cd Anal
	$ cp suiso.mp3 ~/.suiso.mp3
	$ cp push.mp3 ~/.push.mp3
	$ echo "alias ls='afplay ~/.suiso.mp3 & ls'" >> ~/.bash_profile
	$ echo "alias cd='afplay ~/.push.mp3 & cd'" >> ~/.bash_profile
	$ echo "alias nano='afplay ~/.push.mp3; afplay ~/.suiso.mp3; nano'" >> ~/.bash_profile
	$ echo "alias vim='afplay ~/.push.mp3; afplay ~/.suiso.mp3; vim'" >> ~/.bash_profile
	$ echo "alias emacs='afplay ~/.push.mp3; afplay ~/.suiso.mp3; emacs'" >> ~/.bash_profile
```
